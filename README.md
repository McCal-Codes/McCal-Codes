Much of my work explores how **structured systems can support creative expression**.

## Current Projects

### TerraNova

A large-scale procedural world generation project for **Hytale**.

TerraNova explores how terrain systems, biome architecture, and environmental signals can create worlds that feel natural but still carry narrative.

Focus areas:

- biome architecture
- terrain density systems
- node-based generation workflows
- environmental storytelling
- modding infrastructure

---

### Abridged

A minimalist **news reader designed to reduce doomscrolling**.

The goal is to present complex stories in a way that is:

- clear
- structured
- calm to read
- respectful of the reader's attention

---

### McCal Media Platform

My personal **custom-coded web platform**.

Originally built to replace Squarespace, the project evolved into a modular system built around reusable UI components and design tokens.

Key ideas behind the system:

- modular widget architecture
- design-token based UI
- portfolio and podcast integration
- CDN-hosted components
